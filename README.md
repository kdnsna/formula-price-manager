# 全员全产品计价公式管理台

本地单页工具，用于维护全员全产品计价公式的月度变更台账、下月公式建议、公式解释、年度规则版本、25→26 单价切换、单位口径和新增配置记录。

## 功能

- 月度公式助手：按当前月份和目标月份批量生成下月公式草案、含义说明和复核提示。
- 月度变更台账：按月份登记状态、备注和变更后公式，可一键采纳月度建议。
- 公式库：查看原公式、公式描述、风险等级和 2026 配置建议。
- 详情抽屉：公式说明、单价切换、单位口径、月度参数、变更历史。
- 新增新配：将 2026 新增或拆分指标一键生成本月配置记录。
- 年度版本：保留年度规则版本，并生成年度迁移任务。
- 导入导出：支持 Excel 导入，支持导出 Excel 台账和 JSON 备份。

## 本地运行

```bash
npm install
npm run dev -- --host 127.0.0.1
```

打开：

```text
http://127.0.0.1:5174/
```

## 构建

```bash
npm run build
```

## 数据说明

应用内置了首版 Excel 种子数据，月度变更记录、月度建议和年度迁移任务保存在浏览器 `localStorage`，key 为 `pricing-formula-manager:v1`。内部 schema 已升级到 v2。

如需从桌面 Excel 重新生成内置种子数据：

```bash
npm run seed
```
